민팅챗봇 ChatGptApi활용

⟨ ⟩

GptApi를 이용해서 간단한 챗봇을 만들었습니다. openAI사이트에서 Api키를 받고 사용하면 됩니다. 다만 유료이기 때문에 확인하고 사용해야하며 18달러까지는 무료입니다. github링크에 ChatGpt.py를 보면 python코드를 사용해서 Spring에 연동해서 사용하고 있습니다. 처음에는 자이썬을 이용해서 연결해보려 했지만 오류도 많고 원하는 방식이 아니라는것을 알고 ProcessBuilder를 사용해 파이썬을 작동 했습니다. 주요 코드만 보고 설명하겠습니다. String pythonScriptPath = "ChatGpt.py경로" ProcessBuilder processBuilder = new ProcessBuilder("python", pythonScriptPath, nickname, apiKey); processBuilder.redirectErrorStream(true); Process process = processBuilder.start(); 파이썬 경로를 읽어와서 ProcessBuilder로 읽는 형식으로 작동시켰으며 nickname, apikey는 스프링에서 값을 받아 파이썬으로 전달하기위해서 processBuilder에 작성했습니다. 파이썬에서 작동시키는 것이기 떄문에 타임리프가 아닌 tkinker을 사용해서 창을 띄워 대화창을 만들었습니다. (mintingController, ChatGpt.py, footer.html)

시세조회

크롤링을 이용해서 시세조회를 할 수 있게 만들었습니다. GptApi를 사용했던 방식과 똑같이 파이썬을 실행해서 데이터를 구하고 데이터를 스프링으로 연동시켜 타임리프에서 보이도록 만들었습니다. 값을 주고 받기 위해서는 Json형식으로 주고 받아야 했고 Json을 바로 타임리프로 보이게 하기위해 Ajax를 사용했습니다. # 데이터를 JSON 형식으로 출력 json_data = json.dumps(data, ensure_ascii=False) sys.stdout.buffer.write(codecs.encode(json_data, 'utf-8')) sys.stdout.flush() $.ajax({ url: '/lowest', type: 'GET', data: { inputValue: inputValue }, success: function (data) { $('#mintyResultBody').empty(); data.forEach(function (item) { var row = $(''); var titleLink = $('').attr('href', '/boardDetail/' + item.id).text(item.title); $('').append(titleLink).appendTo(row); $('').text(item.price).appendTo(row); row.appendTo('#mintyResultBody'); }); } }); 위와 같은 형식으로 데이터를 주고 받았고 정상 출력되는 것을 확인했습니다. 크롤링 자체는 별로 어렵지 않았으나 데이터를 주고 정상출력되게 하는것이 더 어려웠습니다.
